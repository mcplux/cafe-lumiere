# Café Lumière

# Run in development

1. Raise API: `https://github.com/mcplux/cafe-lumiere-api`

2. Clone the project
```
git clone https://github.com/mcplux/cafe-lumiere.git
```

3. Install dependencies

```
npm install
```

4. Duplicate `.env.template` file as `.env`

```
cp .env.template .env
```

5. Run application
```
npm run dev
```
